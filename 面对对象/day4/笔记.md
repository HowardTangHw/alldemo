### 绑定数据的几种方式

1. +豆豆加加法则 

```javascript
"+this.name+"
```

2. 使用formateString绑定数据

> 首先寻找@(name)这种格式的数据,然后用真是的数据替换掉@(name) 

```javascript
//简单使用
var user ={name:"111"};
var span =document.getElementById("span");
var str = '欢迎@(name)访问本网站';
str = formateString(str,user);
span.innerHTML=str;
```

> formateString这个函数只能针对简单的对象进行绑定(局限性)
